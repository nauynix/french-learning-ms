# Multi Search French

This [multisearch site](https://nauynix.github.io/french-learning-ms/Multisearch-French.html) searches words in French to help an English speaker build flash cards by searching:
- Linguee: For translation and example
- French Wiktionary: For word sounds
- Google Images: Get images to choose to build the flash card

The site will open up 3 new tabs in the browser with search results from each of the sites for the word searched.
